# Kausa – Sitio Web Oficial

Sitio web corporativo de **Kausa Marketing & Comunicaciones**.

## Tecnologías

- HTML5 semántico
- CSS3 (variables, grid, animaciones)
- JavaScript vanilla (sin dependencias)
- Fuentes: Cormorant Garamond + DM Sans (Google Fonts)

## Estructura del proyecto

```
kausa/
├── index.html          # Página principal
├── assets/
│   ├── css/
│   │   └── style.css   # Todos los estilos
│   ├── js/
│   │   └── main.js     # Lógica e interacciones
│   └── img/
│       ├── kausa_logo.png      # Logo completo
│       └── kausa_logo_k.PNG    # Isotipo (K)
└── README.md
```

## Secciones

1. **Hero** – Propuesta de valor principal
2. **Prueba social** – Métricas clave con animación count-up
3. **Problema** – Conexión con los dolores del cliente
4. **Nosotros** – Historia y diferenciación
5. **Servicios** – Estrategia, Performance, Contenido, Web, Growth
6. **¿Por qué Kausa?** – Propuesta de valor diferencial
7. **Proyectos** – Casos de éxito con métricas
8. **Diagnóstico gratuito** – Lead magnet
9. **Contacto** – Formulario + WhatsApp

## Desarrollo local

Abrir `index.html` directamente en el navegador. No requiere servidor ni dependencias.
